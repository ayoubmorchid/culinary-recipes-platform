# Culinary Recipes Platform

Full Stack recipe sharing platform built with Spring Boot and React.

## Features

- User authentication with JWT
- User profiles
- Recipe CRUD
- Categories
- Comments
- Ratings
- Favorites
- Admin dashboard
- File uploads

## Tech Stack

### Backend
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- MySQL

### Frontend
- React
- React Router
- Axios
- Context API

## Run Backend

```bash
cd backend
mvn spring-boot:run


cd frontend
npm install
npm run dev
